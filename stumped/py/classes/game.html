Game¶

class games.stumped.game.Game¶

The class representing the Game in the Stumped game.

Gather branches and build up your lodge as beavers fight to survive.

beavers¶

Every Beaver in the game.

Return type:list[Beaver]
current_player¶

The player whose turn it is currently. That player can send commands. Other players cannot.

Return type:Player
current_turn¶

The current turn number, starting at 0 for the first player’s turn.

Return type:int
free_beavers_count¶

When a Player has less Beavers than this number, then recruiting other Beavers is free.

Return type:int
game_objects¶

A mapping of every game object’s ID to the actual game object. Primarily used by the server and client to easily refer to the game objects via ID.

Return type:dict[str, GameObject]
get_tile_at(x, y)¶

Gets the Tile at a specified (x, y) position :param x: integer between 0 and the mapWidth :type x: int :param y: integer between 0 and the mapHeight :type y: int

Returns:the Tile at (x, y) or None if out of bounds
Return type:Tile
jobs¶

All the Jobs that Beavers can have in the game.

Return type:list[Job]
lodge_cost_constant¶

Constant number used to calculate what it costs to spawn a new lodge.

Return type:float
lodges_to_win¶

How many lodges must be owned by a Player at once to win the game.

Return type:int
map_height¶

The number of Tiles in the map along the y (vertical) axis.

Return type:int
map_width¶

The number of Tiles in the map along the x (horizontal) axis.

Return type:int
max_turns¶

The maximum number of turns before the game will automatically end.

Return type:int
players¶

List of all the players in the game.

Return type:list[Player]
session¶

A unique identifier for the game instance that is being played.

Return type:str
spawner¶

Every Spawner in the game.

Return type:list[Spawner]
spawner_harvest_constant¶

Constant number used to calculate how many branches/food Beavers harvest from Spawners.

Return type:float
spawner_types¶

All the types of Spawners in the game.

Return type:list[str]
tiles¶

All the tiles in the map, stored in Row-major order. Use x + y * mapWidth to access the correct index.

Return type:list[Tile]
