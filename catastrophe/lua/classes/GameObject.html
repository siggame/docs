Class GameObject

An object in the game.

The most basic class that all game classes should inherit from automatically.

Fields

self.gameObjectName String representing the top level Class that this game object is an instance of.
self.id A unique id for each instance of a GameObject or a sub class.
self.logs Any strings logged will be stored here.

Methods

GameObject:log (message) Adds a message to this GameObject's logs.


Fields

self.gameObjectName
String representing the top level Class that this game object is an instance of. Used for reflection to create new instances on clients, but exposed for convenience should AIs want this data.
self.id
A unique id for each instance of a GameObject or a sub class. Used for client and server communication. Should never change value after being set.
self.logs
Any strings logged will be stored here. Intended for debugging.

Methods

GameObject:log (message)
Adds a message to this GameObject's logs. Intended for your own debugging purposes, as strings stored here are saved in the gamelog.

Parameters:

  • message string A string to add to this GameObject's log. Intended for debugging.
