Fire Department¶

class games.anarchy.FireDepartment¶

The class representing the FireDepartment in the Anarchy game.

Can put out fires completely.

extinguish(building)¶

Bribes this FireDepartment to extinguish the some of the fire in a building.

Args:
building (Building): The Building you want to extinguish.
Returns:
bool: true if the bribe worked, false otherwise
fire_extinguished¶

The amount of fire removed from a building when bribed to extinguish a building.
