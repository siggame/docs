std::string Chess::Piece::move ( std::string  rank,
int  file,
std::string  promotionType = "" 
)

Moves the piece from its current location to the given rank and file.

Parameters
rankThe rank (x) coordinate to move to. Must be [a-h].
fileThe file (y) coordinate to move to. Must be [1-8].
promotionTypeIf this is a Pawn moving to the end of the board then this parameter is what to promote it to.
Returns
The standard algebraic notation (SAN) representation of the move if successful, empty string otherwise. In addition if you fail your move you lose.
