Class Game

Use cowboys to have a good time and play some music on a Piano, while brawling with enemy Cowboys.

Fields

self.name The name of this game, "{game_name}".
self.bartenderCooldown How many turns a Bartender will be busy for after throwing a Bottle.
self.bottles All the beer Bottles currently flying across the saloon in the game.
self.brawlerDamage How much damage is applied to neighboring things bit by the Sharpshooter between turns.
self.cowboys Every Cowboy in the game.
self.currentPlayer The player whose turn it is currently.
self.currentTurn The current turn number, starting at 0 for the first player's turn.
self.furnishings Every furnishing in the game.
self.gameObjects A mapping of every game object's ID to the actual game object.
self.jobs All the jobs that Cowboys can be called in with.
self.mapHeight The number of Tiles in the map along the y (vertical) axis.
self.mapWidth The number of Tiles in the map along the x (horizontal) axis.
self.maxCowboysPerJob The maximum number of Cowboys a Player can bring into the saloon of each specific job.
self.maxTurns The maximum number of turns before the game will automatically end.
self.players List of all the players in the game.
self.rowdinessToSiesta When a player's rowdiness reaches or exceeds this number their Cowboys take a collective siesta.
self.session A unique identifier for the game instance that is being played.
self.sharpshooterDamage How much damage is applied to things hit by Sharpshooters when they act.
self.siestaLength How long siestas are for a player's team.
self.tiles All the tiles in the map, stored in Row-major order.
self.turnsDrunk How many turns a Cowboy will be drunk for if a bottle breaks on it.
