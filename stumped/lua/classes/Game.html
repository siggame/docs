Class Game

Gather branches and build up your lodge as beavers fight to survive.

Fields

self.name The name of this game, "{game_name}".
self.beavers Every Beaver in the game.
self.currentPlayer The player whose turn it is currently.
self.currentTurn The current turn number, starting at 0 for the first player's turn.
self.freeBeaversCount When a Player has less Beavers than this number, then recruiting other Beavers is free.
self.gameObjects A mapping of every game object's ID to the actual game object.
self.jobs All the Jobs that Beavers can have in the game.
self.lodgeCostConstant Constant number used to calculate what it costs to spawn a new lodge.
self.lodgesToWin How many lodges must be owned by a Player at once to win the game.
self.mapHeight The number of Tiles in the map along the y (vertical) axis.
self.mapWidth The number of Tiles in the map along the x (horizontal) axis.
self.maxTurns The maximum number of turns before the game will automatically end.
self.players List of all the players in the game.
self.session A unique identifier for the game instance that is being played.
self.spawner Every Spawner in the game.
self.spawnerHarvestConstant Constant number used to calculate how many branches/food Beavers harvest from Spawners.
self.spawnerTypes All the types of Spawners in the game.
self.tiles All the tiles in the map, stored in Row-major order.

A mapping of every game object's ID to the actual game object. Primarily used by the server and client to easily refer to the game objects via ID.

self.spawner
Every Spawner in the game.
self.spawnerHarvestConstant
Constant number used to calculate how many branches/food Beavers harvest from Spawners.
self.spawnerTypes
All the types of Spawners in the game.
