Class Tile

A Tile in the game that makes up the 2D map grid.

Fields

self.beaver The Beaver on this Tile if present, otherwise nil.
self.branches The number of branches dropped on this Tile.
self.flowDirection The cardinal direction water is flowing on this Tile ('North', 'East', 'South', 'West').
self.food The number of food dropped on this Tile.
self.lodgeOwner The owner of the Beaver lodge on this Tile, if present, otherwise nil.
self.spawner The resource Spawner on this Tile if present, otherwise nil.
self.tileEast The Tile to the 'East' of this one (x+1, y).
self.tileNorth The Tile to the 'North' of this one (x, y-1).
self.tileSouth The Tile to the 'South' of this one (x, y+1).
self.tileWest The Tile to the 'West' of this one (x-1, y).
self.type What type of Tile this is, either 'Water' or 'Land'.
self.x The x (horizontal) position of this Tile.
self.y The y (vertical) position of this Tile.
self.gameObjectName (inherited) String representing the top level Class that this game object is an instance of.
self.id (inherited) A unique id for each instance of a GameObject or a sub class.
self.logs (inherited) Any strings logged will be stored here.
Tile.directions The valid directions that tiles can be in, "North", "East", "South", or "West"
