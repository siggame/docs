Class Game

There's an infestation of enemy spiders challenging your queen broodmother spider! Protect her and attack the other broodmother in this turn based, node based, game.

Fields

self.name The name of this game, "{game_name}".
self.currentPlayer The player whose turn it is currently.
self.currentTurn The current turn number, starting at 0 for the first player's turn.
self.cutSpeed The speed at which Cutters work to do cut Webs.
self.eggsScalar Constant used to calculate how many eggs BroodMothers get on their owner's turns.
self.gameObjects A mapping of every game object's ID to the actual game object.
self.initialWebStrength The starting strength for Webs.
self.maxTurns The maximum number of turns before the game will automatically end.
self.movementSpeed The speed at which Spiderlings move on Webs.
self.nests Every Nest in the game.
self.players List of all the players in the game.
self.session A unique identifier for the game instance that is being played.
self.spitSpeed The speed at which Spitters work to spit new Webs.
self.weavePower How much web strength is added or removed from Webs when they are weaved.
self.weaveSpeed The speed at which Weavers work to do strengthens and weakens on Webs.
self.webs Every Web in the game.
