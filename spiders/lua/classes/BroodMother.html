Class BroodMother

The Spider Queen.

She alone can spawn Spiderlings for each Player, and if she dies the owner loses.

self.eggs How many eggs the BroodMother has to spawn Spiderlings this turn.

Methods

BroodMother:consume (spiderling)
Consumes a Spiderling of the same owner to regain some eggs to spawn more Spiderlings.

Parameters:

  • spiderling Spiderling The Spiderling to consume. It must be on the same Nest as this BroodMother.

Returns:

    bool True if the Spiderling was consumed. False otherwise.
BroodMother:spawn (spiderlingType)
Spawns a new Spiderling on the same Nest as this BroodMother, consuming an egg.

Parameters:

  • spiderlingType string The string name of the Spiderling class you want to Spawn. Must be 'Spitter', 'Weaver', or 'Cutter'.

Returns:

    Spiderling The newly spwaned Spiderling if successful. Null otherwise.
