Piece¶

class games.chess.Piece¶

Bases: games.chess.game_object.GameObject

The class representing the Piece in the Chess game.

A chess piece

captured¶

When the piece has been captured (removed from the board) this is true. Otherwise false.

file¶

The file (y) coordinate of the checker represented as a number [1-8].

game_object_name¶

String representing the top level Class that this game object is an instance of. Used for reflection to create new instances on clients, but exposed for convenience should AIs want this data.

has_moved¶

If the piece has moved from its starting position.

id¶

A unique id for each instance of a GameObject or a sub class. Used for client and server communication. Should never change value after being set.

log(message)¶

adds a message to this game object’s log. Intended for debugging purposes.

Args:
message (str): A string to add to this GameObject’s log. Intended for debugging.
logs¶

Any strings logged will be stored here when this game object logs the strings. Intended for debugging.

move(rank, file, promotionType='')¶

Moves the piece from its current location to the given rank and file.

Args:
rank (str): The rank (x) coordinate to move to. Must be [a-h]. file (int): The file (y) coordinate to move to. Must be [1-8]. promotion_type (Optional[str]): If this is a Pawn moving to the end of the board then this parameter is what to promote it to.
Returns:
str: The standard algebraic notation (SAN) representation of the move if successful, empty string otherwise. In addition if you fail your move you lose.
owner¶

The player that controls this chess Piece.

rank¶

The rank (x) coordinate of the checker represented as a letter [a-h].

type¶

The type of chess piece this is, either: ‘King’, ‘Queen’, ‘Knight’, ‘Rook’, ‘Bishop’, or ‘Pawn’.
