Detailed Description

A basic AI interface.

This class implements most the code an AI would need to interface with the lower-level game code. AIs should extend this class to get a lot of builer-plate code out of the way The provided AI class does just that.

Member Function Documentation

abstract bool BaseAI.run ( )
pure virtual

This is run every turn . Return true to end the turn, return false to request a status update from the server and then immediately rerun this function with the latest game status.
