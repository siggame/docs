Array.<Tile> findPath(start, goal)

ai.js, line 227
A very basic path finding algorithm (Breadth First Search) that when given a starting Tile, will return a valid path to the goal Tile.
Name Type Description
start Tile the starting Tile
goal Tile the goal Tile
Returns: Array.<Tile>
array of Tiles representing the path, the the first element being a valid adjacent Tile to the start, and the last element being the goal.
