Chess::Move * Chess::Piece::move ( std::string  file,
int  rank,
std::string  promotionType = "" 
)

Moves the Piece from its current location to the given rank and file.

Parameters
fileThe file coordinate to move to. Must be [a-h].
rankThe rank coordinate to move to. Must be [1-8].
promotionTypeIf this is a Pawn moving to the end of the board then this parameter is what to promote it to. When used must be 'Queen', 'Knight', 'Rook', or 'Bishop'.
Returns
The Move you did if successful, otherwise null if invalid. In addition if your move was invalid you will lose.
